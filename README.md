# Lumière Estética — Landing Page

Landing page desenvolvida para clínica de estética, com foco em design sofisticado e conversão.

## Tecnologias

- HTML5
- CSS3 (variáveis, animações, grid, responsivo)
- JavaScript puro

## Funcionalidades

- Layout responsivo (mobile e desktop)
- Animações de entrada com scroll reveal
- Cursor personalizado
- Navbar com efeito ao rolar
- Contador de estatísticas animado
- Strip de serviços com marquee
- Mapa integrado com Google Maps
- Links diretos para WhatsApp e Instagram

## Estrutura

```
├── index.html
├── style.css
└── script.js
```
